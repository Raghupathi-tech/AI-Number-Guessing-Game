# AI Number Guessing Game

## Description
A Java application where the system generates a random number and provides intelligent hints to help users guess the correct number.

## Features
- Random number generation
- Intelligent hints
- User interaction
- Win detection

## Technologies Used
- Java
- Random Class
- Scanner Class

## How to Run
1. Open the project in Eclipse.
2. Run NumberGuessAI.java.
3. Guess the generated number.
